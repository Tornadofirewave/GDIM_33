# GDIM 33 In-Class Activities
## W4
### Activity 1

The current mechanics playable in my build are:

- Pickupable ability to allow the player to throw a star once pressing X.
- Basic movement controls including left, right, and jump.
- Dialogue and exploration states
- Interactable dialogue by pressing up arrow while in front of an NPC
- Concluding dialogue upon repeated interactions with an NPC
- Custom-made physics system with player movement

My goals for this playtest is for the playtesters to experience a smooth, easy to navigate movement system that feels like they're in full control of the player. I want the sensation of the core mechanics of being able to throw a star and moving around to feel fun and natural.
